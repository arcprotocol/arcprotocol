## Reporting a Vulnerability

The ARC Protocol team takes security vulnerabilities seriously. We appreciate your efforts to responsibly disclose your findings and will address security issues quickly.

### How to Report a Security Vulnerability

To report a security issue, please email moein.roghani@proton.me
